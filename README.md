# Auto-Start Calculator App

A simple **Python Calculator** that automatically launches at Windows startup using **Task Scheduler**. This project demonstrates how to combine GUI programming with Python and automate tasks on Windows.  

---

## Features

- Basic calculator with addition, subtraction, multiplication, and division
- GUI built with **Tkinter**
- Automatic startup using Windows **Task Scheduler**
- Works as a Python script or packaged executable
